Employee Attendance System

Description
Employee Attendance System is an application designed to automate employee attendance tracking. The system allows for recording attendance, absences, and tardiness, as well as displaying reports for each employee. The project demonstrates the application of object-oriented programming (OOP) principles using the Java programming language.

Project Requirements List
Below are the 10 key functionalities of the system:
1. Employee registration in the system with basic details (ID, name, position).
2. Recording employees' clock-in and clock-out times.
3. Calculating tardiness and overtime.
4. Validating data during updates.
5. Storing attendance information using collections.
6. Generating reports for each employee.
7. Displaying a list of all employees and their attendance status.
8. Managing data through a console interface.
9. Handling errors and exceptions.
10. Optimizing the system using OOP principles (inheritance, encapsulation, polymorphism).
